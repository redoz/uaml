What you can do

  • Draw data marts and joinable relationships on a Miro-like canvas, with a field-level ERD view (columns, primary keys, join keys).
  • Start from templates for real domains: E-commerce, SaaS, Marketplace, Marketing / Lead-gen, Mobile / Gaming, Finance, Healthcare, plus public datasets (Bitcoin, Stack Overflow).
  • Import and export Open Knowledge Format (OKF) — a vendor-neutral Markdown bundle for data models, so your model stays portable and never locked in.
  • Generate a model with AI: copy a short brief into Claude, ChatGPT or Gemini and import what it returns. See the AI build guide.
  • Get Insight Questions: the business questions a model can answer, generated from schema metadata only — no row-level data leaves your browser.
  • Share a model via a link that reopens your exact diagram, and push it into OWOX Data Marts as drafts.

Frequently asked

Is it free? Yes — the whole canvas is free and works without signing in. An OWOX API key is only needed to push a model into OWOX Data Marts.

What is OKF? Open Knowledge Format is a vendor-neutral Markdown format for data models (a folder of .md files with YAML frontmatter). OWOX Model Canvas imports and exports OKF bundles so your model is portable across tools and git.

Do I need an account? No. You can design models, use templates, get AI question suggestions, and share a link anonymously.
